## Lance Locallement

Clonez le projet

```bash
  git clone https://github.com/SebMZI/Portfolio_Sophie_Bluel.git
```

Rendez-vous sur le projet

```bash
  cd Portfolio_Sophie_Bluel
```

Installez les dépéndances

```bash
  npm install
```

Lancez le serveur

```bash
  npm start
```
